# Building WET Compliant Forms

As of v2, WBR supports the full set of
[WET form controls](https://wet-boew.github.io/wet-boew-styleguide/design/forms-en.html#selects)
and built-in validation.

For validation, WBR supports [specialized validation](https://wet-boew.github.io/v4.0-ci/docs/ref/formvalid/formvalid-en.html)
using the `data-rule-xxx` attributes however these attributes must always be
lowercase. For example, `data-rule-postalCodeCA` must be changed to
`data-rule-postalcodeca`.

Use a `data-msg` attribute to override the default validation message.
